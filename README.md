# LEGO Collector Management System

A personal app to scan LEGO sets, track inventory and value, and suggest builds based on owned pieces.

## Planned Features
- Scan LEGO box barcodes to log sets
- Track collection value over time
- Suggest builds from owned inventory

## Tech Stack
- Frontend: React
- Backend: Python
- Hosting: AWS S3 + CloudFront
- Database: DynamoDB (future)
